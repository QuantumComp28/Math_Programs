# **Math Function Scripts**
These are a few small mathematical python scripts I have created in my free time. One preforms Riemann integration to approximate the value of a given integral
and graphs that function with its riemann approximation (given by small summed up areas of rectangles). 
The other one breaks apart any number into its basic prime factorization (which is all the primes which when multipled together give you the number originally 
plugged in.)
